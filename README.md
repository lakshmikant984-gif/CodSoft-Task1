Overview of the Project

Project:Rule-Based Chatbot Using Python

Objective

The main objective of this project is to develop a simple chatbot that interacts with users by providing predefined responses based on their inputs. It helps in understanding the basic concepts of Artificial Intelligence, Natural Language Processing (NLP), and conversation flow using Python.

Key Activities

Design a simple chatbot interface.
Accept user input from the keyboard.
Compare the input with predefined rules using if-else statements.
Generate appropriate responses based on the user's query.
Handle unknown inputs with a default response.
Continue the conversation until the user enters "bye" to exit.

Technologies Used

Programming Language: Python
Concepts: Rule-Based AI, If-Else Statements, String Handling
Libraries: Built-in Python functions (optionally datetime for date/time)
Development Environment: Visual Studio Code, IDLE, PyCharm, or any Python IDE
Operating System: Windows, Linux, or macOS
